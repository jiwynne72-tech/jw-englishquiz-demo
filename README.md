# Time to Quiz: English Games 🎓
## SAMPLE VERSION FOR TESTING
**Time to Quiz** is an interactive Progressive Web App (PWA) designed to make learning English engaging and fun. It features five distinct game modes that test vocabulary, grammar, intuition, and cultural knowledge.

### 🚀 [Play the Game Now!](https://YOUR-USERNAME.github.io/YOUR-REPO-NAME/)

---

### 🎮 Game Modes
* **The Wall:** Categorize groups of words that share hidden connections.
* **Connections:** Decipher the link between four increasingly specific clues.
* **Chronology:** Match historical events to the correct year.
* **Odd One Out:** Identify the word that doesn't belong in the group.
* **English Sayings:** Master common idioms and their meanings.

---

### 📱 How to Install (PWA)
This app is a **Progressive Web App**, meaning you can install it on your device and use it offline without visiting an App Store.

* **On iOS (Safari):** Tap the **Share** button (square with an arrow) and select **"Add to Home Screen."**
* **On Android (Chrome):** Tap the **three dots** in the corner and select **"Install app"** or **"Add to Home Screen."**
* **On Desktop:** Click the **Install** icon on the right side of the browser's address bar.

### 🛠️ Built With
